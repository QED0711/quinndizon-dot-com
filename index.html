Git Checklist

An app to help you plan and manage your git workflow by creating and checking off small, managable tasks.

Simply drag and drop any git initialized directory below. If it doesn't already have a .gitchecklist file, you will be prompted to add one. You can then add your tasks to the .gitchecklist file in the format - '[ ] task name'; When that file is loaded into this application, you can check off each task as you complete them. As you complete tasks,and check them off, your changes will be automatically committed. You can optionally have your commits pushed to your branch of choice.
